# IRIS-MD
---
IRIS-MD is a flexible tool for generating Markdown-based quizzes from structured JSON. It supports:

- Dynamic templating
- Math formulas using `$$...$$`
- Collapsible answers and markschemes
- Manual or computed answers
- Easy export or in-memory use

## Installation

```bash
npm install iris-md
